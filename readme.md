#phpqa

###how to use
```
project_path="/var/www/your_project/"
phpqa_tool="/var/www/tool/phpqa/vendor/bin/phpqa"
cd $your_project
$phpqa_tool --ignoredDirs build,vendor,node_modules,composer --report
```

###look all tools
```sh
./vendor/bin/phpqa tools
```
